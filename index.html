Unshift

This method adds elements to the beginning of an array

let arr = [2, 3, 4];
arr.unshift(1); // Adds 1 at the beginning

Shift

This method removes the first element from an array

let arr = [1, 2, 3, 4]; let firstElement = arr.shift(); // Removes the first element (1)

Split

This method divides a string into an ordered list of substrings, puts these substrings into an array. The division is done by searching for a key that matches where the splits should occur

let str = "Hello World";
let words = str.split(" "); // Splits the string into an array of words
console.log(words); // Output: ["Hello", "World"]

Example:

let CFGDegrees = ['Data Science', 'Software Engineering'];
CFGDegrees.unshift('FullStack Engineering');
console.log(CFGDegrees); // Output: ['FullStack Engineering', 'Data Science', 'Software Engineering']
let CFGDegrees = ['FullStack Engineering', 'Data Science', 'Software Engineering'];
CFGDegrees.shift(); console.log('CFGDegrees'); //Output: ['Data Science', 'Software Engineering']

Onclick
